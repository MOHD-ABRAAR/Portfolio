Identification of Bird Calls

I have developed a web application designed to track and identify birds based on their vocalizations. The application takes a bird's sound as input and accurately predicts its name and species, making it particularly useful for monitoring and preserving endangered bird species. For development, I utilized Python as the primary programming language along with Jupyter Notebook, and I applied various deep learning algorithms. To evaluate the system's performance, I employed metrics such as Random Forest, Recall, F1 Score, SVM, and Decision Tree, achieving an accuracy rate of approximately 98%.
